Importing a vWii NAND Into Dolphin

Section I - Performing a NAND Backup

If you already made a NAND backup of your Wii U, you can safely skip this section.

  1. Download the latest version of Tiramisu.
  2. Extract the .zip file and copy the wiiu folder to the root of your console's SD card.
    3. Allow your computer to merge/overwrite anything if prompted.
  3. Reinsert the SD card into your console.
  4. Enter the PayloadLoader on your Wii U.
    5. If you have autobooting configured, hold B when powering it on.
    If you do not have autobooting configured, hold B when launching the Health & Safety app.
  5. Select the nanddumper payload.
  6. Use the GamePad's D-Pad to select SLCCMPT and OTP, then press A to begin dumping.
    7. The dumping process may take a few minutes.
  7. Power off your console and reinsert its SD card into your computer.
    8. Copy slccmpt.bin and otp.bin onto your computer.
  8. After copying, feel free to delete these files from your SD card to save space.

Section II - Extracting Your vWii's Keys

  1. Download the latest version of ShowMiiVWiiKeys.
  2. Extract the appropriate executable for your computer's operating system from .zip file.
  3. Copy your otp.bin into the same directory as the ShowMiiVWiiKeys executable and run it.
  4. There should now be two new files: keys.bin and keys.txt. If not, ensure you are using the correct platform executable and have otp.bin in the same directory.

Section III - Importing the NAND Backup

  1. If you haven't already, install the latest beta version of Dolphin.
    2. Windows users will also need to install the 64-bit Visual C++ Redistributable. Check Dolphin's download page for a link.
  2. Make sure Dolphin is not running, and navigate to its user directory.
    3. For Windows: The default location is Dolphin Emulator in your Documents folder.
    For macOS: The default location is ~/Library/Application Support/Dolphin.
    For Linux: The default location is determined by the XDG Base Directory Specification, which can vary between distributions (see this page for more info). Helpful tip: if the directory ~/.dolphin-emu exists, it will be used instead.
  3. Create a new folder named Wii in this directory.
    4. If one exists already, rename it to WiiBackup. For the following steps to work, the Wii directory must be empty. Save data will be restored from this backup folder in a later section.
  4. Launch Dolphin.
  5. Navigate to "Options" > "Configuration" > "General".
  6. Enable "Allow Mismatched Region Settings".
    7. This will prevent Dolphin from possibly generating a new console identity when an out-of-region game is launched, which would reset the 7-day waiting period on Wiimmfi.
  7. Close the settings window and return to Dolphin's main interface.
  8. Select "Tools" > "Manage NAND" > "Import BootMii NAND Backup...".
    9. Select your slccmpt.bin file as the BootMii NAND backup file.
    Select your keys.bin file as the BootMii keys file.

Section IV - Repairing System Titles

  1. Download, extract, and run NUS Downloader.
    2. macOS/Linux users may need to run this either with Wine or in a virtual machine.
  2. Select "Database..." > "System" > "System Menu", and choose "Latest Version".
    3. Ensure that the "Pack WAD" box is checked on the main app interface.
  3. Press "Start NUS Download!", and wait for it to finish.
  4. Select "Database..." > "System" > "Mii Channel" > "41 (All/System)" > "Latest Version".
    5. Ensure that "Pack WAD" is still checked.
  5. Press "Start NUS Download!", and wait for it to finish.
  6. In your system's file explorer, navigate to the folder you extracted NUS Downloader into and verify that these WAD files downloaded correctly. The version numbers were the latest at the time of writing and may vary.
    7. Wii Menu: titles/0000000100000002/518/RVL-WiiSystemmenu-v518.wad
    Mii Channel: titles/0001000248414341/6/RVL-NigaoeNR-v6.wad
  7. Close NUS Downloader and return to Dolphin.
  8. Select "Tools" > "Install WAD..."
  9. Select your RVL-WiiSystemmenu-v###.wad file to install the Wii Menu.
  10. Repeat the previous two steps for RVL-NigaoeNR-v#.wad to install the Mii Channel.
  11. Select "Tools" > "Perform Online System Update" > "Current Region".

Section V - Restoring Existing Dolphin Save Data

This section will replace the save data imported from your NAND backup with your previous Dolphin save data. If you wish to instead keep the save data from your vWii's NAND, you can safely skip this section.
